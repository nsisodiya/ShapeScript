ShapeScript Tutorial

Learn to generate 3D models with code

1. The Core Philosophy

ShapeScript is a lightweight browser modeling tool. Instead of clicking and dragging shapes, you write standard JavaScript code. Every script must return the final 3D model that you want to render and export to STL.

Note: The default unit of measurement is millimeters (mm), standard for 3D slicers.

2. 3D Primitives

Create basic solids using the following global functions:

Cubes and Boxes

cube(size) creates a symmetric cube. box(width, depth, height) creates a cuboid. They are anchored in the positive octant, stretching from 0 to the defined dimension on all axes.

// A 20mm x 20mm x 20mm cube
return cube(20);

// A rectangular box (Width=40mm, Depth=30mm, Height=10mm)
return box(40, 30, 10);

Spheres

sphere(radius) creates a sphere centered at the origin [0, 0, 0].

// A sphere of radius 15mm (diameter 30mm)
return sphere(15);

Cylinders and Cones

Both are centered on the XY plane and extend along the positive Z axis from z = 0 to z = height.

// Cylinder with radius 10mm and height 40mm
return cylinder(10, 40);

// Cone tapering to a point: top radius 0, bottom radius 15mm, height 30mm
return cone(0, 15, 30);

// Truncated cone (top radius 8mm, bottom radius 15mm, height 30mm)
return cone(8, 15, 30);

Torus

torus(radius, tubeRadius) creates a ring centered at the origin lying flat on the XY plane.

// Major ring radius 25mm, tube thickness radius 4mm
return torus(25, 4);

3. Chainable Transformations

Apply moves, rotations, scaling, or mirroring by chaining methods directly on any geometry object.

Moving (Translation)

Shift a shape by offset values along X, Y, and Z axes:

// Move a 20mm cube by 15mm on X, and 5mm on Z
return cube(20).move(15, 0, 5);

Rotating

Rotate a shape around X, Y, and Z axes. Angles are specified in degrees:

// Rotate a box 45 degrees around the Z axis (yaw)
return box(30, 10, 10).rotate(0, 0, 45);

Scaling

Resize shapes. Passing a single number performs uniform scaling; passing three numbers scales X, Y, and Z independently.

// Double in length on X, shrink by half on Y
return sphere(10).scale(2, 0.5, 1);

Mirroring

Flipped geometry across a coordinate plane ('x', 'y', or 'z'):

// Mirror a sloped shape across the X-axis plane
return cone(0, 10, 20).mirror('x');

4. Boolean Operations

Solid modeling allows you to combine shapes in three ways:

Union

Joins multiple shapes into a single solid body.

const block = box(40, 20, 10);
const peg = cylinder(5, 20).move(20, 10, 0);

return union(block, peg);

Subtract

Cuts tool shapes out of a base shape. Ideal for screw holes, hollow boxes, or slots.

const base = cube(30);
// Drill a 10mm diameter hole through the center of the cube
const drillBit = cylinder(5, 40).move(15, 15, -5);

return subtract(base, drillBit);

Intersect

Finds the common volume shared by all shapes.

return intersect(
  cube(30),
  sphere(20).move(15, 15, 15)
);

5. Dynamic Parametric Controls

You can create sliders, checkboxes, and select menus that draw in the controls panel on-the-fly. Adjusting these controls rebuilds the model instantly.

  • slider(name, default, min, max): Returns a number.
  • checkbox(name, default): Returns `true` or `false`.
  • select(name, default, optionsArray): Returns the selected string option.
// Register inputs
const length = slider("Width", 50, 20, 100);
const wall = slider("Thickness", 4, 2, 8);
const hollow = checkbox("Hollow Out", true);

const outer = box(length, 40, 30);

if (hollow) {
  const inner = box(length - wall*2, 40 - wall*2, 30)
    .move(wall, wall, wall);
  return subtract(outer, inner);
}

return outer;

6. Advanced Modeling: Loops and Math

Because ShapeScript is pure JavaScript, you can use loops, variables, and trigonometry to create complex mechanical or decorative items.

Example: Circular Pattern (Gear Teeth)

const teethCount = 12;
const coreRadius = 20;
const thick = 8;

const gearCore = cylinder(coreRadius, thick);
let gear = gearCore;

for (let i = 0; i < teethCount; i++) {
  // Angle for this tooth in degrees
  const angle = (i * 360) / teethCount;
  
  // Single tooth cylinder, moved to outer radius, and rotated
  const tooth = box(4, 8, thick)
    .move(-2, coreRadius - 2, 0)
    .rotate(0, 0, angle);
    
  gear = union(gear, tooth);
}

return gear;
